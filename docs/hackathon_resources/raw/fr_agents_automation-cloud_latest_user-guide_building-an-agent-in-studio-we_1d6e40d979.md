 À propos de ce guide

 link Ce guide fournit une vue d&#x27;ensemble complète de l&#x27;initiative UiPath® Agents, conçue pour vous familiariser avec les différents types d&#x27;agents, leurs fonctionnalités et les avantages qu&#x27;ils apportent à vos projets d&#x27;automatisation.

 Le contenu est structuré pour couvrir deux domaines principaux :

- L&#x27;expérience native des agents UiPath : les sections dédiées décrivent en détail les outils et les méthodes intégrés, adaptés aux utilisateurs de tous les niveaux de compétence, permettant de créer des agents low-code directement dans UiPath Platform™.

- Agents codés : cette section se concentre sur les agents codés créés en externe qui peuvent être intégrés à l&#x27;écosystème UiPath. Elle concerne les utilisateurs plus avancés et les scénarios d&#x27;automatisation complexes.

 En explorant les deux aspects, ce guide vise à vous doter d’une compréhension approfondie de l’infrastructure de l’agent UiPath, ce qui vous permet d’exploiter tout le potentiel des Agents dans vos workflows d’automatisation.

 Terminologie ​

 link
 Dans ce guide, nous utilisons les termes suivants :

- Agents UiPath  : désigne les agents natifs low-code créés au sein de la plateforme UiPath.

- Agents codés ou agents externes  : désigne les agents (basés sur du code ou low-code) développés en dehors d&#x27;UiPath, mais déployables sur la plateforme.
