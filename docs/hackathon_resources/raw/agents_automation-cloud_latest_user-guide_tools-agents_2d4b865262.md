 Agents

 link An agent can invoke other agents published in your tenant, letting you compose more complex workflows by chaining agent capabilities together.

 To add an agent as a tool:

- From the Definition panel, under the Tools section, select Add tool .

- Select the Agents tab and search for agents using the search bar or select one from the available list.

- Ensure the agent you want to use is published and deployed as a process to a shared Orchestrator folder. Only deployed agents can be used as tools.

- Once added, the agent appears as a tool with its input and output schema automatically exposed, including any file-based arguments.

- Use the system prompt to describe the agent’s purpose and specify when it should be invoked within the workflow. This helps the calling agent determine when and how to use the tool effectively.
