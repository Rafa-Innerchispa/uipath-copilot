 About this guide

 link This guide provides a comprehensive overview of the UiPath® Agents initiative, designed to familiarize you with the various agent types, their capabilities, and the benefits they bring to your automation projects.

 The content is structured to cover two main areas:

- The native UiPath Agents experience: The dedicated sections detail the built-in tools and methods for creating low-code agents directly within the UiPath Platform™, suitable for users of all skill levels.

- Coded agents: This section focuses on externally built, code-based agents that can be integrated into the UiPath ecosystem, catering to more advanced users and complex automation scenarios.

 By exploring both aspects, this guide aims to equip you with a thorough understanding of the UiPath agent framework, enabling you to leverage the full potential of agents in your automation workflows.

 Terminology ​

 link
 Throughout this guide we are using the following terms:

- UiPath Agents : Refers to the native, low-code agents built within the UiPath Platform.

- Coded Agents or External Agents : Refers to agents (code-based or low-code) developed outside UiPath but deployable on the platform.
