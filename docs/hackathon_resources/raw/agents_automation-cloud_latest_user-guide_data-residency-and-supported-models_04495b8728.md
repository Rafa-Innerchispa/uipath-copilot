 Data residency and supported models

 link Service availability can vary across UiPath’s global cloud regions. When a model or its capacity is unavailable in a given region, requests are automatically routed to the nearest region with available capacity. For more information, refer to the Data residency and AI features and model routing sections of the Automation Cloud Admin guide.
