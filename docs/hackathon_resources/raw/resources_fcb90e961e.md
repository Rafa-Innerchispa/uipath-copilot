 Start here &nbsp;

 UiPath AgentHack is built around real, working agentic systems, n ot concepts or prototypes. &nbsp;

 You’ll design and ship a solution on the UiPath Platform that: &nbsp;

-
 coordinates multiple agents&nbsp; &nbsp;

-
 handles long-running processes&nbsp; &nbsp;

-
 handles failures, retries, and exception paths gracefully &nbsp;

-
 maintains visibility, auditability, and control across the workflow &nbsp;

-
 keeps humans in the loop at critical decision points&nbsp; &nbsp;

-
 Register on Devpost &nbsp;

-

-

 What is UiPath Labs? &nbsp;

 UiPath Labs is the sandbox environment where you'll build, test, and run your AgentHack project. It comes pre-configured with the UiPath tools and services you need, so your team can focus on building, not setup. &nbsp;

 Core tools &amp; technologies &nbsp;

 All solutions must run on the UiPath Automation Cloud. You can include Agent Builder, Maestro, API Workflows, coding agents, and RPA where needed. You're welcome to bring in agents built on external frameworks and LLMs, in fact, we encourage it. The point is that UiPath is the orchestration and governance layer that ties everything together. &nbsp;
