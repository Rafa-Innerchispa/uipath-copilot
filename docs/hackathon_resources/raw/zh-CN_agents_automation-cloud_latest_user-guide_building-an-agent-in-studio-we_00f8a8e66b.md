 关于本指南

 link 本指南提供 UiPath™ 智能体计划的全面概述，旨在帮助您熟悉各种智能体类型、其功能及其为您的自动化项目带来的优势。

 其内容结构主要涵盖两个方面：

- 原生 UiPath 智能体体验：这个部分专门详细介绍了用于直接在 UiPath Platform™ 中创建低代码智能体的内置工具和方法，适合所有技能级别的用户。

- 编码智能体：这部分重点介绍可以集成到 UiPath 生态系统的外部构建、基于代码的智能体，以满足更高级用户和复杂自动化场景的需求。

 本指南旨在通过探索这两个方面来帮助您全面了解 UiPath 智能体框架，以便您在自动化工作流中充分发挥智能体的潜力。

 术语 ​

 link
 在本指南中，我们使用了以下术语：

- UiPath 智能体 ：指在 UiPath Platform 中构建的原生低代码智能体。

- 编码智能体 或 外部智能体 ：指在 UiPath Platform 外部开发，但可在平台上部署的智能体（基于代码或低代码）。
